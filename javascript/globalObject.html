Objetos Globais

São objetos padrões que estão sempre disponíveis para o usuário. Os Objetos Padrãoes não são estruturas padrões como o while ou if.

Exemplos de Objetos Padrões do Javascript:

  • Object
  • Array
  • Date
  • String
  • Number
  • Math

Por definição da Linguagem Javascript os objetos padrões sempre começam com uma letra maiúscula. Um objeto padrão tem várias propriedades e métodos já definidos na própria linguagem.
